How it works
Three seconds.
Zero friction.
01
♟
Finish your game
Play on Chess.com as usual. The extension runs in the background — no setup needed mid-game.
02
🔵
Click the blue button
When the result modal appears, a new "Analyze in Chesskit" button shows up alongside the usual options.
03
⚡
Auto-loaded & ready
Chesskit opens in a new tab with your username filled in and the latest game already selected. Just click Analyze.
Features
Built for speed.
🎯
Zero configuration
The extension auto-detects your Chess.com username from the page. No accounts, no API keys.
⚡
Instant redirect
Opens Chesskit, fills your username, and selects the latest game automatically. Saves ~30 seconds every game.
🔒
Privacy first
No data sent anywhere. Your username is stored locally in Chrome. The extension only reads what's on the page.
🆓
Completely free
Open source, MIT licensed. No ads, no tracking, no premium tier. Just a small tool that does one thing well.
Installation
Up and running
in 60 seconds.
1
Download and unzip chesskit-extension.zip from the Releases page
2
Open Chrome and navigate to chrome://extensions/
3
Enable Developer mode in the top-right toggle
4
Click Load unpacked and select the unzipped folder
5
Play a game on Chess.com — the blue button will appear when it ends ✓
