How it works
Three seconds.
Zero friction.
01
♟
Finish your game
Play on Chess.com as usual. The extension runs silently in the background.
02
🔵
Click the blue button
When the result screen appears, a new "Analyze in Chesskit" button shows up alongside the usual options.
03
⚡
Auto-loaded & ready
Chesskit opens in a new tab with your latest game already selected. Click Analyze and you're in.
Features
Built for speed.
🎯
Zero configuration
Auto-detects your Chess.com username from the page. No accounts, no API keys.
⚡
Instant redirect
Opens Chesskit, fills your username, and selects the latest game automatically. Saves ~30 seconds every game.
🔒
Privacy first
No data sent anywhere. Your username is stored locally in Chrome only.
🆓
Completely free
Open source, MIT licensed. No ads, no tracking, no premium tier.
Installation
Up and running
in 60 seconds.
1
Download and unzip chesskit-extension.zip from the Releases page
2
Open Chrome and navigate to chrome://extensions/
3
Enable Developer mode using the top-right toggle
4
Click Load unpacked and select the unzipped folder
5
Play a game on Chess.com — the blue button appears when it ends ✓
